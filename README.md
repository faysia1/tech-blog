# 14 Model-View-Controller (MVC): Tech Blog

## User Story

```md
AS A developer who writes about tech
I WANT a CMS-style blog site
SO THAT I can publish articles, blog posts, and my thoughts and opinions
```
```

# Installation
I used javascript, Node.js, and Heroku.

# Usage
The purpose of this site is to view posts on a tech blog and create/edit your own posts. You can also make comments
